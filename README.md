# Tic Tac Toe Game

This is a simple Tic Tac Toe game created using HTML, CSS, and JavaScript.

## Features
- Two player game (X and O)
- Winner detection
- Draw detection
- Restart button

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder.
2. Open `index.html` in your browser.
3. Play the game.

## Rules
- Player X starts first.
- Get 3 marks in a row (horizontal, vertical, or diagonal) to win.
- If all boxes are filled and no one wins, the game is a draw.
